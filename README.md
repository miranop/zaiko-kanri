# 📦 在庫管理システム (Zaiko Kanri)

Inventory Management System - Claude Code で vibe coding を使って作成された在庫管理ソフトウェア

## 🎯 概要

この在庫管理システムは、商品の在庫状況を簡単に管理できるWebアプリケーションです。
Node.js + Express + SQLite で構築された、シンプルで使いやすいインターフェースを持っています。

## ✨ 機能

- ✅ **商品管理**: 商品の登録・編集・削除
- 📊 **在庫追跡**: リアルタイムでの在庫数の管理
- ⚠️ **在庫アラート**: 最小在庫数を下回る商品の自動検出
- 🔢 **在庫調整**: 簡単な在庫の増減操作
- 💰 **価格管理**: 商品価格の設定
- 🏷️ **カテゴリー分類**: 商品のカテゴリー管理
- 🔍 **フィルタリング**: 全商品表示/在庫不足商品表示の切り替え

## 🚀 セットアップ

### 必要条件

- Node.js (v14以上)
- npm

### インストール

1. リポジトリをクローン:
```bash
git clone https://github.com/miranop/zaiko-kanri.git
cd zaiko-kanri
```

2. 依存パッケージをインストール:
```bash
npm install
```

3. サーバーを起動:
```bash
npm start
```

4. ブラウザで開く:
```
http://localhost:3000
```

## 📖 使い方

### 商品の登録

1. 左側のフォームに商品情報を入力
2. 必須項目: 商品名、在庫数
3. オプション項目: SKU、説明、最小在庫数、価格、カテゴリー
4. 「登録」ボタンをクリック

### 商品の編集

1. 右側の商品一覧から編集したい商品の「✏️ 編集」ボタンをクリック
2. フォームに情報が読み込まれるので、編集
3. 「更新」ボタンをクリック

### 在庫調整

1. 商品カード下部の「±数量」欄に調整値を入力
   - プラス値で在庫を増加
   - マイナス値で在庫を減少
2. 「在庫調整」ボタンをクリック

### 在庫不足の確認

- 「在庫不足」ボタンをクリックすると、最小在庫数を下回る商品のみが表示されます
- 在庫不足の商品は黄色の枠で強調表示されます

## 🔌 API エンドポイント

### 商品管理

- `GET /api/products` - 全商品を取得
- `GET /api/products/:id` - 特定の商品を取得
- `POST /api/products` - 新規商品を作成
- `PUT /api/products/:id` - 商品情報を更新
- `DELETE /api/products/:id` - 商品を削除

### 在庫管理

- `PATCH /api/products/:id/quantity` - 在庫数を調整
- `GET /api/products/alerts/low-stock` - 在庫不足の商品を取得
